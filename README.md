# AI-Based Sentiment Analyzer

## Overview:
This project provides a sentiment analysis tool powered by Hugging Face’s pre-trained model. It allows users to analyze sentiment (positive, negative, neutral) from product reviews or social media posts.



## 🛠️ Technologies Used

- Python + Flask
**MongoDB** (for storing sentiment analysis results)
- **HTML/CSS** (Frontend)
- MongoDB Atlas
- Hugging Face Transformers
- Chart.js
- pdfkit + wkhtmltopdf + multilingual fonts
- Render (Deployment)

## 🚀 Features

- 🔍 Real-time Sentiment Detection (via XLM-RoBERTa)
- 🌍 Multilingual Text Input (e.g., English, Hindi, Bengali, Italian, Japanese, etc.)
- 🧾 PDF Report Generation with correct fonts
- 📊 Dashboard with Charts
- 💾 MongoDB Atlas Integration for Persistent Storage
- ☁️ Deployed on Render.com
